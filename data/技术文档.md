# 垂直领域专家助手技术文档

## 系统概述

本系统是一个基于 RAG（检索增强生成）技术的智能问答系统，专门用于垂直领域的知识问答服务。

### 核心技术栈

1. **LlamaIndex**: 用于构建高效的向量检索和查询引擎
2. **HuggingFace**: 使用 bge-small-zh-v1.5 模型进行中文文本向量化
3. **DeepSeek API**: 提供强大的大语言模型支持
4. **Streamlit**: 快速构建交互式Web界面

## 主要功能

### 1. 知识库管理
- 自动读取 data/ 目录下的所有 .txt 和 .md 文件
- 智能文本分块处理
- 基于内存的向量索引构建

### 2. 智能检索
- 将用户问题转换为向量表示
- 检索 Top-K=3 的最相关内容
- 结合检索到的上下文生成回答

### 3. 对话交互
- 支持多轮对话
- 流式输出提高用户体验
- 聊天历史记录保存

## 技术特点

### 向量嵌入
使用 BAAI/bge-small-zh-v1.5 模型进行中文文本向量化，该模型具有以下优势：
- 轻量级设计，适合本地部署
- 中文语义理解能力强
- 支持多种文本格式

### 检索策略
采用基于余弦相似度的向量检索策略：
- 相似度阈值过滤
- Top-K 选择机制
- 上下文窗口动态调整

### 生成优化
- Prompt 优化设计
- 温度参数控制输出多样性
- 响应时间优化

## 使用指南

### 启动应用
```bash
# 安装依赖
pip install -r requirements_web.txt

# 启动应用
streamlit run web_app.py
```

### 配置说明
1. 在侧边栏输入 DeepSeek API Key
2. 或者使用模拟层模式（无需 API Key）
3. 点击"重新加载知识库"按钮
4. 开始提问

### 最佳实践
1. 提问时尽量清晰具体
2. 可以追问以获取更详细的信息
3. 定期更新知识库文档
4. 使用模拟层进行测试

## 常见问题

### Q: 如何添加新的知识库文档？
A: 直接将 .txt 或 .md 文件放到 data/ 目录下，然后点击"重新加载知识库"按钮。

### Q: 系统支持哪些语言？
A: 主要支持中文，使用专门的中文嵌入模型。

### Q: 如何提高回答质量？
A: 提供更多高质量的知识库文档，提问时尽量明确具体。
