## 📊 系统指标

### 性能
- 向量化速度: ~5-10 docs/sec (CPU)
- 检索延迟: < 100ms
- 生成延迟: 取决于 API 响应

### 资源占用
- 内存: ~2-4GB (包括模型)
- 磁盘: ~500MB (模型文件)
- CPU: 中等负载
